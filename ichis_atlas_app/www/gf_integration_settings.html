Google Gemini
Integração com a API Gemini do Google para geração de texto, análise de documentos e assistência inteligente baseada em IA. Requer uma API Key gerada no Google AI Studio (aistudio.google.com).
Enable Gemini Integration Ativa ou desativa toda a integração com o Gemini
Autenticação
Chave de API gerada no Google AI Studio. Mantenha em segurança — concede acesso total à sua cota de uso. Deixe em branco para manter o valor atualmente salvo.
Parâmetros do Modelo
Modelo padrão utilizado nas chamadas à API. O gemini-2.0-flash oferece melhor custo-benefício para uso geral; gemini-1.5-pro para tarefas mais complexas.
Controla a criatividade das respostas. 0.0 = determinístico e preciso; 2.0 = mais variado e criativo. Recomendado: 0.7 para uso geral.
Número máximo de tokens gerados na resposta. Aproximadamente 4 caracteres por token. Valores maiores aumentam o custo por chamada.
Instrução de sistema enviada em todas as chamadas para definir o comportamento, papel e contexto do modelo. Use para personalizar respostas ao domínio da GREENFARMS.

OpenAI ChatGPT
Integração com a API da OpenAI para geração de texto, análise e automação com os modelos GPT. Requer uma API Key gerada no painel da OpenAI (platform.openai.com).
Enable OpenAI Integration Ativa ou desativa toda a integração com a OpenAI
Autenticação
Chave de API gerada no painel da OpenAI. Começa com "sk-proj-" ou "sk-". Deixe em branco para manter o valor atualmente salvo.
ID da organização OpenAI. Necessário apenas se sua conta pertencer a múltiplas organizações e você precisar especificar qual usar.
ID do projeto OpenAI para isolamento de uso e controle de billing por projeto dentro da organização.
Parâmetros do Modelo
Modelo GPT padrão. O gpt-4o-mini oferece melhor custo-benefício; gpt-4o para tarefas que exigem maior raciocínio e qualidade.
Controla a aleatoriedade das respostas. 0.0 = mais preciso e determinístico; 2.0 = mais variado e criativo. Recomendado: 0.7.
Limite máximo de tokens na resposta (inclui tokens do prompt + resposta). Verificar limite do modelo escolhido.
Mensagem de sistema enviada no início de todas as conversas para definir o comportamento, papel e contexto do assistente.
